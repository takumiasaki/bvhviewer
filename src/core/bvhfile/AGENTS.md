# BVH File Core Design Document

## 目的

src/core/bvhfile 以下は BVH ファイルのパースとデータ管理を担当する層となる。
BVH ファイルのファイルのパースとデータ管理を担当する層となる。

## 要件

* **F01: BVHファイルの読み込み**
    * アプリケーションがBVHファイルを読み込むことができること。
    * 読み込んだファイルの内容（階層構造、モーションデータ）を正確にパースし、内部データ構造に格納できること。
* **F02: データ管理**
    * 読み込んだBVHファイルのデータを効率的に管理できること。
    * 階層構造やモーションデータを適切なデータ構造で表現し、必要に応じてアクセスできること。
    * データはスケルトン部とモーション部に分けて管理すること。
* **F03: エラーハンドリング**
    * 不正なBVHファイルやパースエラーが発生した場合に、適切なエラーメッセージを提供し、アプリケーションがクラッシュしないようにすること。

## ステータス

* 実装済み
* バグ修正のみ可

## スコープ

* BVH ファイルのパースとデータ管理を担当する層
* BVH 以外のモーションフォーマットには対応しない
* 3D 表示の直接的な実装は含まない、純データ層とする

## データ構造

データ構造は docs/design/data-structure.md を参照。

## 将来の拡張

* バグ修正以外の拡張は行わない
* バグ修正時はユニットテストの追加も実施する事
